隐私政策

欢迎使用智伴成长!

我们深知个人信息对您的重要性,并会尽全力保护您的个人信息安全可靠。我们致力于维持您对我们的信任,恪守以下原则,保护您的个人信息:权责一致原则、目的明确原则、选择同意原则、最少够用原则、确保安全原则、主体参与原则、公开透明原则等。

一、我们如何收集和使用您的个人信息

1.1 账号注册与登录

不需要账号即可体验功能

1.2 语音咨询功能

智伴成长的核心功能是基于AI的语音咨询服务。为了提供此服务,我们会:

  • 录制语音:当您使用语音咨询功能时,我们会录制您的语音输入
  • 本地识别:语音识别在您的设备本地完成(使用SenseVoice技术),音频文件不会上传到服务器
  • 文字传输:仅将识别后的文字内容发送到服务器,用于AI分析和生成建议
  • 情绪检测:识别您的情绪状态(如平静、焦虑、愤怒等),以提供更贴心的建议
隐私保护承诺: 您的语音录音仅在设备本地处理,不会上传到任何服务器。我们仅收集识别后的文字内容用于AI分析。

1.3 成长日记功能

当您使用成长日记功能时,我们会收集:

  • 日记内容:您记录的文字或语音转换后的文字
  • 情绪标签:AI自动识别的情绪标签
  • 时间信息:日记创建和修改时间
  • AI建议:系统为您生成的育儿建议

1.4 习惯养成功能

当您使用习惯养成功能时,我们会收集:

  • 习惯任务:您为孩子设置的习惯任务(如刷牙、阅读)
  • 打卡记录:孩子完成任务的时间和频率
  • 积分信息:孩子获得的积分和兑换记录

1.5 设备权限

为了提供完整的功能体验,我们需要获取以下设备权限:

  • 麦克风权限:用于录制语音,进行语音咨询和日记记录
  • 语音识别权限:用于将语音转换为文字
  • 网络访问权限:用于与服务器通信,获取AI建议
权限说明: 您可以在设备的系统设置中随时管理这些权限。拒绝授权某些权限可能会影响相关功能的使用。

二、我们如何使用Cookie和同类技术

智伴成长是一款原生iOS应用,不使用Cookie技术。我们使用以下本地存储技术:

  • Core Data:用于本地存储您的日记、习惯记录等数据
  • UserDefaults:用于存储应用设置和用户偏好
  • Keychain:用于安全存储登录凭证

三、我们如何共享、转让、公开披露您的个人信息

3.1 共享

我们不会与任何第三方公司、组织和个人共享您的个人信息,除非:

  • 获得您的明确同意
  • 根据法律法规或政府主管部门的要求
  • 为了保护您或他人的生命、财产等重大合法权益

3.2 第三方服务

我们使用以下第三方服务提供商:

  • 阿里云通义千问:用于AI对话生成。我们仅向其发送文字内容(不包含您的个人身份信息),用于生成育儿建议
  • PostgreSQL数据库:用于存储您的账号信息、日记和习惯记录,部署在我们自己的服务器上
重要说明: 我们已与第三方服务提供商签订严格的数据保护协议,要求他们按照本隐私政策以及其他相关的保密和安全措施来处理个人信息。

3.3 转让

我们不会将您的个人信息转让给任何公司、组织和个人,除非:

  • 获得您的明确同意
  • 在涉及合并、收购或破产清算时,如涉及到个人信息转让,我们会要求新的持有您个人信息的公司、组织继续受本隐私政策的约束

3.4 公开披露

我们不会公开披露您的个人信息,除非:

  • 获得您的明确同意
  • 基于法律法规、法律程序、诉讼或政府主管部门强制性要求

四、我们如何保护您的个人信息

4.1 数据安全技术措施

  • 传输加密:使用HTTPS协议加密传输数据
  • 存储加密:敏感信息在数据库中加密存储
  • 访问控制:严格限制数据访问权限,仅授权人员可访问
  • 本地处理:语音识别在设备本地完成,音频不上传服务器

4.2 数据安全管理措施

  • 建立数据分类分级制度,对不同级别的数据采取不同的保护措施
  • 定期进行安全审计和漏洞扫描
  • 对员工进行数据安全培训,签订保密协议
  • 制定应急响应预案,及时处理安全事件

4.3 安全事件处置

如不幸发生个人信息安全事件,我们将按照法律法规的要求,及时向您告知:安全事件的基本情况和可能的影响、我们已采取或将要采取的处置措施、您可自主防范和降低风险的建议等。

五、您的权利

5.1 访问和更新您的个人信息

您可以在应用内查看和修改您的个人信息:

  • 在"我的"页面查看和修改昵称
  • 在"日记"页面查看、编辑或删除您的成长日记
  • 在"习惯"页面管理习惯任务和打卡记录

5.2 删除您的个人信息

您可以通过以下方式删除您的个人信息:

  • 删除单条日记记录
  • 删除习惯任务
  • 注销账号(将删除所有个人信息)

5.3 撤回同意

您可以通过以下方式撤回授权:

  • 在iOS系统设置中关闭麦克风、语音识别等权限
  • 停止使用相关功能
  • 注销账号

5.4 注销账号

您可以通过以下方式注销账号:

  1. 联系我们的客服邮箱:support@ailiaobar.com
  2. 提供您的账号信息以验证身份
  3. 我们将在15个工作日内完成账号注销

注意: 注销账号后,我们将删除您的所有个人信息,且无法恢复。

六、未成年人保护

智伴成长主要面向2-8岁儿童的家长。我们重视未成年人的个人信息保护:

  • 如果您是未成年人的监护人,请您指导和监督未成年人使用我们的服务
  • 我们不会主动收集未成年人的个人信息
  • 如果我们发现在未事先获得可证实的监护人同意的情况下收集了未成年人的个人信息,会设法尽快删除相关数据

七、隐私政策的更新

我们可能会适时修订本隐私政策的条款。当隐私政策发生变更时:

  • 我们会在应用内通过弹窗、推送通知等方式向您展示变更内容
  • 如果变更会导致您在本隐私政策下权利的实质减少,我们将在变更生效前通过显著方式提示您
  • 您继续使用我们的服务,即表示同意受经修订的隐私政策的约束
